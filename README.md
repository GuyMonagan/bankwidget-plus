# bankwidget-plus

Виджет для отображения замаскированных номеров карт и счетов.

Включает функции маскировки, базовую фильтрацию и сортировку, а также тесты.  
Проект создан с использованием **Poetry**.

## Цель проекта

- Маскировка номеров карт и счетов (скрытие части символов)
- Фильтрация транзакций по статусу (`filter_by_state`)
- Сортировка по дате транзакций (`sort_by_date`)
- Подготовка к интеграции с веб-интерфейсом (в будущем)

## Установка

```bash
git clone https://github.com/GuyMonagan/bankwidget-plus.git
cd bankwidget-plus
poetry install
```

## Запуск тестов

```
poetry run pytest

```
